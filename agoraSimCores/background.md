## Background

- Define um fundo para o nosso elemento
- Sua área de atuação
- Por padrão, é transparente

### Exemplos

- Usar cores sólidas 
- Usar imagens
- Controlar
  - A posição das imagens
  - Se elas se reperem ou não
  - O tamanho delas na caixa
- Usar cor e imagem juntas
- Usar cor gradiente
